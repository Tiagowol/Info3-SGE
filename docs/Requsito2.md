#02 - Cadastro de Turma
COMO Coordenador Pedagógico,

QUERO cadastrar uma nova turma

PARA que eu possa organizar os alunos por grupos e anos letivos estruturados.

Critérios de Aceitação:

Deve ser possível informar o nome da turma (ex: Informática 3V) e o código (ex: INF3V-2026).

O sistema deve impedir o cadastro de turmas com códigos duplicados.

Uma turma nova deve iniciar com a lista de alunos vazia.
