#05 - Cadastro de Atividade Escolar
COMO Organizador de Eventos ou Professor,

QUERO cadastrar uma atividade no sistema

PARA que os alunos saibam quais tarefas obrigatórias ou extracurriculares precisam cumprir.

Critérios de Aceitação:

Deve ser possível informar o nome da atividade e o seu tipo (ex: Extracurricular, Obrigatória).

O sistema deve validar que tanto o nome quanto o tipo foram preenchidos.
