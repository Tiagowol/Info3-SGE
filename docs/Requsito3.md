#03 - Vinculação de Aluno à Turma
COMO Secretário Escolar,

QUERO matricular/vincular um aluno em uma turma

PARA que ele faça parte da lista oficial de chamada daquele ano letivo.

Critérios de Aceitação:

O sistema deve permitir selecionar um aluno já cadastrado e adicioná-lo à lista de alunos de uma turma.

O sistema deve impedir que o mesmo aluno seja adicionado mais de uma vez na mesma turma.
