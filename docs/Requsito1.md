#01 - Cadastro de Aluno
COMO Secretário Escolar ou Coordenador,

QUERO cadastrar um novo aluno no sistema

PARA que eu possa manter o registro dos dados pessoais e acadêmicos dos estudantes atualizado.

Critérios de Aceitação:

Deve ser possível informar o nome, a matrícula e a idade do aluno.

O sistema deve impedir o cadastro de alunos com o mesmo número de matrícula.

O sistema deve validar que a idade do aluno é maior que zero.
